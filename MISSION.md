# Mission

**Pass the Claude Certified Architect – Foundations certification** and use it to
land or advance a job building production applications with Claude.

## Why (the real driver)
- The user wants the credential as a **job/career lever** ("land/advance a job").
- Starting point: **new to Claude/the Anthropic API** — little hands-on API work yet.
- So the mission is dual: (1) pass the exam, (2) genuinely acquire the architect
  skill the exam validates — *making informed tradeoff decisions when building
  real solutions with Claude*.

## The exam (verified from the official page, 2026-07-13)
- Provider: **Anthropic partner program**, hosted on Skilljar.
- URL: https://anthropic-partners.skilljar.com/claude-certified-architect-foundations-certification
- Cost: **$125** (purchase required to sit the exam).
- Positioning: validates that a practitioner "can make informed decisions about
  tradeoffs when implementing real-world solutions with Claude."
- **Four core domains:**
  1. Claude API
  2. Model Context Protocol (MCP)
  3. Claude Agent SDK
  4. Claude Code
- Exact question count / time limit / passing score: **live in the Exam Guide PDF**
  behind the Skilljar login — user to download and share so we can tune format.

## How we're teaching to it
- Learning style: **hands-on tasks, worked examples, concept maps, quizzes/recall** (all four).
- Emphasis on **tradeoff decisions** (the exam's stated core), not rote memorization.
- Sequence: API foundation → tool use → MCP → Agent SDK → Claude Code → mixed review.
