# 0001 — Mission and exam target established

**Date:** 2026-07-13
**Status:** Active

## Context
User arrived asking to "get certified as Claude Certified Architect." Initial web
checks showed no generic Anthropic "Architect" exam, so I flagged the risk of teaching
to a phantom target. User then supplied the real URL.

## What was decided / learned
- **Real exam confirmed:** Claude Certified Architect – Foundations, Anthropic partner
  program on Skilljar, $125. Four domains: Claude API, MCP, Claude Agent SDK, Claude Code.
  Core stated skill = *making informed tradeoff decisions in real-world Claude solutions.*
- **Learner is new** to the Anthropic API; mission is a **career lever** (job).
- **Learning style:** hands-on tasks + worked examples + concept maps + quizzes (all four).
- Exact exam format (Q count, time, pass %) still unknown — in the Exam Guide PDF the
  user needs to download.

## Zone of proximal development (starting point)
- Cannot assume any API fluency. Start at domain 1 (Claude API) fundamentals.
- Bias every lesson toward *tradeoff reasoning* (which surface / which setting / when),
  since that's what the exam grades — not memorization of syntax or model IDs.

## Implications for future sessions
- Sequence: API → tool use → MCP → Agent SDK → Claude Code → mixed exam-style review.
- Teach concepts that survive model churn (adaptive thinking, effort, caching, the
  harness/deployment split) rather than volatile model-id strings.
- Get the Exam Guide PDF to calibrate quiz format to the real thing.
